TROUBLESHOOTING

Common Issues

Homebrew fails: Run /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Python packages fail: Create fresh venv: python3 -m venv ~/experiment_env && source ~/experiment_env/bin/activate
Remote access denied: Enable in System Preferences ‚Üí Sharing ‚Üí Remote Login
Port already in use: Find process: lsof -i :5000 then kill it
